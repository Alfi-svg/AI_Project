Offline rule: URL will NOT download. It only works if the video already exists in backend/storage/url_cache/

Offline URL Cache Instructions

  1. Paste your URL in the box.
  2. If system says “not found in cache”, it will show a URL key.
  3. Put your downloaded video into backend/storage/url_cache/
  4. Rename it to <URL_KEY>.mp4 (or .mkv/.avi)
  5. Run again. It will process fully offline.
